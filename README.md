# AssetLift CRM

Fresh standalone CRM codebase for AssetLift Lending.

## Current scope

- login shell
- dashboard command center
- contacts
- pipeline
- inbox
- calendar
- documents
- automations
- reporting
- integrations

## Product direction

This build is meant to replace the earlier wrong-repo CRM path.
It is structured as a separate app so it can grow toward Genie Rocket-style feature coverage while staying focused on AssetLift lending operations.

## Next implementation layers

- real auth
- database and persistence
- borrower and lender detail views
- task engine
- email and SMS sync
- calendar sync
- document upload and e-sign workflows
- automation builder
- lender matching and scenario routing
